DNS basics - Google Workspace Admin Help

support.google.com/a/answer/48090?hl=en
To set up Gmail if you have Google Workspace, you need to point your MX records to Google mail servers. Multiple MX records can be defined for a domain, each with a different priority. If mail can't be delivered using the highest priority record, the second priority record is used, and so on.

support.google.com/domains/answer/182076?hl=en
In any web browser, go to admin.google.com. Starting from the sign-in page, enter the email address and password for your admin account (it does not end in @gmail.com).
